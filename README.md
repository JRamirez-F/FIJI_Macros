# FIJI Macros 
 
This repository serves as a central home for various FIJI/ImageJ macros developed for microscopy and image processing tasks. 
It provides descriptions and links to individual macro repositories. 
 
## Available Macros 
 
- FIJI Focus Macros: FIJI_Focus_Macros 
- Numba CLAHE Scripts: FAST_CLAHE 
 
For detailed usage and installation instructions, visit the respective macro repositories.
